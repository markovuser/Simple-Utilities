Simple Utilities - Утилиты для Windows 10, 11.
Бесплатный набор инструментов, предназначенных для компьютеров. 
Это программное обеспечение предоставляет набор инструментов, 
которые включают в себя функции для повышения производительности ПК, 
обеспечения его безопасности и оптимизации процедуры запуска. 
Этот набор инструментов предназначен для всех пользователей ПК от новичков до экспертов. 
Некоторые функции...

Attributes Manager - Программа для изменения атрибутов файлов.
Disk Cleaner - Программа для очистки Windows от мусорных файлов.
File EnCrypt DeCrypt - Программа для шифрования файлов. 
Libraries Manager - Менеджер регистрации библиотек Windows.
Network Manager - Диспетчер управления сетевыми настройками. 
Registry Cleaner - Программа для исправления ошибок в реестре. 
Registry Search - Программа для поиска информации в реестре. 
Startup Manager - Менеджер программ, записанных в автозагрузке.
Uninstall Manager - Программа об установленных программах.
